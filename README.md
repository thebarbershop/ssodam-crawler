# ssodam-crawler

* A web crawler for ssodam.com
* Only for educational, statistic, and entertainment purposes

## How To Run
Just python it.
```
$ python main.py
id: (Give your ID)
password: (Give your password)
```
and it will start crawling from the first page to the last page of the board. 

## External libraries
* requests
* BeautifulSoup4
